6-DOF Robotic Arm – ROS2 Integrated (Work in Progress)
---------------------------------------------------------

A 6-degree-of-freedom robotic arm designed as a final-year project, featuring ROS2 integration and communication between a microcontroller (ESP32) and ROS2 for real-time control and telemetry.
This project is currently in progress, combining mechanical design, embedded systems, and robotics software for precise motion control and system-level experimentation.
